Se trata de un widget de fácil configuración que aparece en la sidebar.
Pensado para todos aquellos que tengan un blog en wordpress, o una template basada en este gestor de contenidos.

== Installation == 
La instalación es sencilla, como cualquier otro plugin de wordpress. Hay que subir el archivo zip que te has descargado.

Si no disponéis de servicio de informática, podéis llamar sin problema a soporte en psmailer para que os guíen en el proceso de instalación o lo hagan ellos mismos si disponen de acceso.

Una vez instalado el widget, tenéis que ir a la sección de Apariencia > Widgets , y ahí arrastrarlo a la barra que deseéis. Cuando esté ahí, solo hay que rellenar el título, el subtítulo, y los datos para vincular el widget con psmailer. El código de API se encuentra en configuración, y el Id de la lista lo veréis en el listado de listas de psmailer.

Una vez todo esté listo, los emails se empezarán a añadir a la lista y ya podréis enviar campañas a todos los espontáneos que se pasen por la web con ganas de recibir vuestros contenidos !




== Frequently Asked Questions ==
= ¿Cómo configuro plugin idioma? =
psmailer utiliza el lenguaje de Dashboard de WP, en ajustes > General. Si el plugin del paquete de idioma no está disponible, el documento se mostrará en inglés.
